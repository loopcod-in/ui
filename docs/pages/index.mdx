# @loopcod/ui

**SaaS-focused UI components for React.**

## What is Loopcod UI?

Loopcod UI is a collection of reusable components and templates designed specifically for building SaaS applications. Unlike generic component libraries, we provide:

- **Complete auth flows** (login, signup, password reset)
- **Dashboard layouts** with navigation
- **Billing components** (pricing tables, checkout)
- **Marketing sections** (hero, features, testimonials)
- **All the UI primitives** you need (button, card, input, etc.)

## Quick Start

```bash
# Initialize your project
npx @loopcod/ui@latest init

# Add components
npx @loopcod/ui@latest add button login-form

# Or use a template
npx @loopcod/ui@latest init --template saas-starter
```

## Features

### 🧩 UI Primitives
Beautiful, accessible components built with Radix UI and Tailwind CSS.

### 🔐 Auth Components
Production-ready authentication forms with validation and error handling.

### 📊 Dashboard
Complete dashboard shell with sidebar, navigation, and responsive layout.

### 💳 Billing
Pricing tables, checkout forms, and subscription management.

### 🎨 Theming
Easy customization with CSS variables. Dark mode included.
