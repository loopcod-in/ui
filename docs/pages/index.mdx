# @loopcod/ui

**SaaS-focused UI components for React.** The component library for building production-ready SaaS applications.

<Callout type="info">
  This is not a component library. It is a collection of re-usable components that you can copy and paste into your apps.
</Callout>

## What is @loopcod/ui?

**@loopcod/ui** provides complete, production-ready SaaS building blocks — not just UI primitives:

- **🔐 Auth Components** — Login forms, signup, password reset (with validation)
- **📊 Dashboard Components** — Shell layout, stats cards, data tables
- **💳 Billing Components** — Pricing tables, checkout forms
- **🎨 Marketing Sections** — Hero, features, testimonials, CTAs
- **🧩 UI Primitives** — Button, Card, Input, and more

## Quick Start

<Steps>

### Create project

```bash
npx create-next-app@latest my-app --typescript --tailwind --eslint
```

### Initialize @loopcod/ui

```bash
cd my-app
npx @loopcod/ui@latest init
```

### Add components

```bash
npx @loopcod/ui@latest add button login-form pricing-table
```

</Steps>
