# Installation

## Requirements

- Node.js 18+
- React 18+
- Tailwind CSS 3.4+

## Quick Start

The fastest way to get started is with our CLI:

```bash
npx @loopcod/ui@latest init
```

This will:
1. Detect your package manager
2. Install required dependencies
3. Set up Tailwind CSS configuration
4. Create `components.json` config file
5. Set up project structure

## Manual Installation

If you prefer manual setup:

### 1. Install dependencies

```bash
npm install class-variance-authority clsx tailwind-merge lucide-react
npm install @radix-ui/react-slot @radix-ui/react-dialog @radix-ui/react-dropdown-menu
```

### 2. Add utility function

Create `lib/utils.ts`:

```ts
import { type ClassValue, clsx } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```

### 3. Configure Tailwind

Update your `tailwind.config.ts` with CSS variables support.

### 4. Add CSS variables

Add to your `globals.css`.
