# Templates

@loopcod/ui provides pre-built templates to get you started quickly.

## Available Templates

### `saas-starter`

Complete SaaS application with auth, billing, and dashboard.

**Includes:**
- Authentication pages (login, signup, forgot password)
- Dashboard with navigation
- Pricing page
- Settings pages
- Pre-configured layout

**Usage:**

```bash
npx @loopcod/ui@latest init --template saas-starter
```

---

### `dashboard`

Admin dashboard template focused on data and analytics.

**Includes:**
- Dashboard layout
- Data tables
- Stats cards
- Filters and search
- Responsive sidebar

**Usage:**

```bash
npx @loopcod/ui@latest init --template dashboard
```

---

## Template Structure

After initializing with a template, your project will have the following structure:

```
my-app/
├── app/
│   ├── (auth)/
│   │   ├── login/
│   │   ├── signup/
│   │   └── forgot-password/
│   ├── (dashboard)/
│   │   ├── layout.tsx
│   │   └── page.tsx
│   ├── layout.tsx
│   └── globals.css
├── components/
│   ├── ui/           # UI primitives
│   ├── auth/         # Auth components
│   ├── dashboard/    # Dashboard components
│   └── billing/      # Billing components
├── lib/
│   └── utils.ts
├── components.json
└── next.config.js
```

---

## Customizing Templates

Templates are starting points. You can:

1. **Add more components**
   ```bash
   npx @loopcod/ui@latest add data-table
   ```

2. **Remove components you don't need**
   Simply delete the files from `components/`

3. **Modify the code**
   Everything is yours to customize!

---

## Creating Your Own Template

Want to create a custom template? Follow these steps:

1. Build your app with @loopcod/ui
2. Create a new folder in `templates/`
3. Copy your app files
4. Update `packages/cli/src/utils/templates.ts`
5. Submit a PR!
